Detailed Description

Represents an access token used for Facebook login, with associated data.

The main object for the SDK, repository for access token, etc.

Container class holding the result of a graph API call - either a JSON object of some type, or an error, depending on the value of the Succeeded property.

Simple class to contain a set of permissions for FB.

Control providing a Facebook Login/Logout button.

class wrapper for FB error responses. For the time being, this is a very bare shell around the error response URL from Facebook, and not a full-blown error class such as exists in the Facebook SDK for iOS.

Class that encapsulates the response from a successful use of the Facebook "App Requests" dialog.
